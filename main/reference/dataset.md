# Data Set

retrieve or set `dataset` slot.

## Usage

``` r
dataset(x)

# S4 method for class 'chevron_tlg'
dataset(x)

dataset(x) <- value

# S4 method for class 'chevron_tlg'
dataset(x) <- value
```

## Arguments

- x:

  (`chevron_tlg`) input.

- value:

  (`character` or `NULL`) defining the dataset slot.

## Value

the `character` stored in the `dataset` slot of the `x` argument.
