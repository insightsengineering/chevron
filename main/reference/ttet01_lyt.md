# `ttet01` Layout

`ttet01` Layout

## Usage

``` r
ttet01_lyt(
  arm_var,
  ref_group,
  summarize_event,
  perform_analysis,
  strata,
  timeunit,
  event_lvls,
  control_survt,
  control_cox_ph,
  control_survtp,
  ...
)
```

## Arguments

- arm_var:

  (`string`) variable used for column splitting

- timeunit:

  (`string`) time unit get from `AVALU`, by default is `"Months"`

- ...:

  not used.
