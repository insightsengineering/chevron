# Set Section Dividers

Set Section Dividers

## Usage

``` r
set_section_div(x)
```

## Arguments

- x:

  (`integerish`) value of at which the section divider should be added.

## Value

invisible `NULL`. Set the `chevron.section_div` option.

## Details

Section dividers are empty lines between sections in tables. E.g. if 1
is used then for the first row split an empty line is added. Currently
it only works for `aet02`, `cmt01a` and `mht01` template.
