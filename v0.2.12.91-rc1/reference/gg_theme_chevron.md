# Theme for Chevron Plot

Theme for Chevron Plot

## Usage

``` r
gg_theme_chevron(
  grid_y = TRUE,
  grid_x = FALSE,
  legend_position = "top",
  text_axis_x_rot = 45
)
```

## Arguments

- grid_y:

  (`flag`) should horizontal grid be displayed.

- grid_x:

  (`flag`) should vertical grid be displayed.

- legend_position:

  (`string`) the position of the legend.

- text_axis_x_rot:

  (`numeric`) the x axis text rotation angle.

## Value

a `theme` object.
