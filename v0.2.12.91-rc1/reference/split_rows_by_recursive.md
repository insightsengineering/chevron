# Count or summarize by groups

Count or summarize by groups

## Usage

``` r
split_rows_by_recursive(lyt, row_split_var, ...)
```

## Arguments

- lyt:

  (`PreDataTableLayouts`) `rtable` layout.

- row_split_var:

  (`character`) variable to split rows by.

- ...:

  Further arguments for `split_rows_by`
