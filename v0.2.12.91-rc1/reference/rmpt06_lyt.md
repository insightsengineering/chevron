# `rmpt06` Layout

`rmpt06` Layout

## Usage

``` r
rmpt06_lyt(
  arm_var,
  lbl_overall,
  method,
  conf_level,
  show_diff,
  ref_group,
  method_diff,
  conf_level_diff,
  grade_groups
)
```

## Arguments

- arm_var:

  (`string`) variable used for column splitting

- lbl_overall:

  (`string`) label used for overall column, if set to `NULL` the overall
  column is omitted
