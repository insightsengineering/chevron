# Count patients recursively

Count patients recursively

## Usage

``` r
count_patients_recursive(lyt, anl_vars, anl_lbls, lbl_vars)
```

## Arguments

- lyt:

  (`PreDataTableLayouts`) `rtable` layout.

- anl_vars:

  Named (`list`) of analysis variables.

- anl_lbls:

  (`character`) of labels.

- lbl_vars:

  Named (`list`) of analysis labels.
