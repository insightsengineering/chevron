# Summary factor allowing NA

Summary factor allowing NA

## Usage

``` r
s_summary_na(
  x,
  labelstr,
  denom = c("n", "N_row", "N_col"),
  .N_row,
  .N_col,
  ...
)
```

## Arguments

- x:

  (`factor`) input.

- denom:

  (`string`) denominator choice.

- .N_row:

  (`integer`) number of rows in row-split dataset.

- .N_col:

  (`integer`) number of rows in column-split dataset.

- ...:

  Not used
