# Helper function to add a column split if specified

Helper function to add a column split if specified

## Usage

``` r
ifneeded_split_col(lyt, var, ...)
```

## Arguments

- lyt:

  (`rtables`) object.

- var:

  (`string`) the name of the variable initiating a new column split.

- ...:

  Additional arguments for `split_cols_by`.

## Value

`rtables` object.
